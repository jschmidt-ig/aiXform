# Bundle contents

1) `Fundamentals-2Week-Syllabus.md`
- Detailed lecture-only syllabus with day plan, session-by-session details, outcomes, and assessment plan.

2) `Deep-Research-Prompt_Slide-Outlines.md`
- Copy/paste prompt template to run in Gemini Deep Research or ChatGPT Deep Research to generate slide-by-slide outlines, demo snippets, and graphics prompts.

3) `Fundamentals-End-Test.md`
- Lecture-only test (10 questions) with critical safety pass/fail and answer key.

4) `Learner-Evaluation-Questionnaire.md`
- Post-module feedback questionnaire (not pass/fail).

5) `Marketing-Page_Fundamentals.html`
- A simple marketing/overview HTML page suitable for sharing internally with outcomes, time commitment, and syllabus-at-a-glance.

## Recommended usage
- Start by running the Deep Research prompt for Sessions 1–2, then iterate until the slide-outline quality matches your teaching style.
- Generate the deck in batches of 3–4 sessions per run to keep outputs consistent and reviewable.
