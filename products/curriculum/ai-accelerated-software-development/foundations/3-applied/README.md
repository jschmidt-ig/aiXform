# Applied Coaching (Weeks 7-10)

Guided coaching where learners apply AI-accelerated development techniques to their actual work projects.

## Overview

- **Duration:** 4 weeks
- **Format:** Hands-on coaching with real projects
- **Prerequisites:** Completion of Fundamentals (Weeks 1-2) and Labs (Weeks 3-6)

## Description

In this phase, learners transition from structured lab exercises to applying AI development techniques on their own production codebases. Instructors provide guidance, code review, and coaching as learners work on real deliverables.
